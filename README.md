# 🚜 Bulldozer-Price-Prediction-Proj.

This machine learning project predicts bulldozer sale prices using historical auction data. The dataset was cleaned and preprocessed by handling missing values, encoding categorical features, and creating date-based features from sale records.

A Random Forest Regressor was trained and optimized using RandomizedSearchCV to improve prediction performance. Model evaluation was performed using metrics such as MAE, RMSLE, and R² score.

### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset Used

## Objective
To build a machine learning model capable of predicting bulldozer sale prices based on historical auction and equipment data.
